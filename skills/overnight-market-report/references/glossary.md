# 용어집 — readability-pass 전용

readability-pass subagent가 입문자용 `-쉬운버전.md`를 만들 때 **이 파일을 Read**해서 전문용어 치환 기준으로 사용합니다. 스킬 본문을 건드리지 않고 용어를 추가/수정할 수 있도록 분리되어 있습니다.

## 사용 규칙

- 한 섹션에서 **처음 등장할 때 괄호 안에 짧은 해설**을 붙이고, 같은 섹션 안에서는 해설 없이 원어만 사용
- 목록에 없는 전문용어가 나오면 같은 방식으로 **한 번만** 괄호 해설 (subagent가 자연스러운 한글 해설을 생성)
- 숫자·티커·URL은 **절대 변경 금지**

## 📘 용어 치환 사전

| 원어 | 괄호 해설 |
|---|---|
| 디버전스 | 서로 다른 방향으로 움직이는 현상 |
| 섹터 로테이션 | 투자금이 업종 간에 이동하는 흐름 |
| 방어주 | 경기와 무관하게 꾸준한 필수소비재·헬스케어·유틸리티 등 |
| 경기민감주 | 경기 좋을 때 잘 오르는 산업재·소재·금융 등 |
| CPI | 소비자물가지수 — 생활물가가 얼마나 올랐는지 보여주는 지표 |
| 근원 CPI | 에너지·식품을 제외한 물가지수, 추세 판단용 |
| PCE | 개인소비지출물가지수 — 연준이 가장 중시하는 물가 지표 |
| FOMC | 연방공개시장위원회 — 미국 기준금리를 정하는 회의 |
| CME FedWatch | 시장이 예상하는 금리 방향을 확률로 보여주는 지표 |
| 10년물 국채금리 | 미국 정부가 10년간 돈을 빌릴 때 내는 이자, 장기금리 기준 |
| 2년물 국채금리 | 단기 정책금리 기대를 반영하는 단기 지표 |
| 수익률 곡선 | 단기·장기 금리 구조. 장기 < 단기면 '역전' — 침체 신호 |
| GDPNow | 애틀랜타 연준이 실시간 추정하는 성장률 |
| VIX | 시장의 공포·변동성을 수치화한 지수, 20 아래면 안정적 |
| ETF | 여러 종목을 묶어 주식처럼 거래하는 상품 |
| 프록시 | 대리로 보는 지표 |
| Mag7 / Magnificent 7 | 빅테크 7인방: Apple·Microsoft·NVIDIA·Google·Amazon·Meta·Tesla |
| 밸류에이션 | 주가가 기업가치 대비 비싼지 싼지 평가 |
| NIM | 예대금리차 — 은행이 대출로 받는 이자와 예금으로 주는 이자의 차이 |
| capex | 설비투자 |
| 차익실현 | 오른 종목을 팔아 수익을 확정하는 행위 |
| 테일리스크 | 가능성은 낮지만 터지면 충격이 큰 위험 |
| 스태그플레이션 | 경기는 멈추고 물가만 오르는 상황 |
| 리플레이션 | 물가가 완만히 오르며 경기가 회복되는 국면 |
| 헤드라인 쇼크 | 표면 수치가 예상보다 나쁘게 나와 시장을 놀라게 하는 것 |
| 호르무즈 해협 | 세계 원유 수송의 약 20%가 지나가는 이란 인근 해협 |
| HBM | 고대역폭 메모리, AI 가속기에 들어가는 고성능 D램 |
| SaaS | 구독형 소프트웨어 사업, 예: Salesforce, ServiceNow |
| 밸류체인 | 제품이 만들어지는 모든 연관 기업 사슬 |
| 강보합 | 살짝 오른 채 마감 |
| 약보합 | 살짝 내린 채 마감 |
| 컨센서스 | 애널리스트들의 평균 예상치 |
| 가이던스 | 기업이 스스로 제시한 실적 전망 |
| 다운그레이드 | 투자의견 하향 |
| 업그레이드 | 투자의견 상향 |
| 크레딧 스프레드 | 회사채와 국채의 금리 차이 — 커지면 부도 우려 ↑ |
| risk-on | 투자자가 위험자산(주식·크립토)을 선호하는 국면 |
| risk-off | 투자자가 안전자산(국채·금·엔화)으로 도망가는 국면 |
| 리플레이션 트레이드 | 물가·경기 회복에 베팅 — 금융·소재·에너지 강세 |
| 방산 ETF | 군수·방위산업체를 묶은 펀드 (ITA 등) |
| 우라늄 ETF | 원자력 연료 관련 기업 묶음 (URA, NLR) |
| 하이일드 | 부도 위험이 있지만 금리가 높은 회사채 (HYG) |
| OPEC+ | 산유국 협의체(OPEC + 러시아 등) — 원유 생산량 조절 |

용어집에 없는 용어가 등장하면 subagent가 위와 같은 "용어 — 한 줄 풀이" 형식으로 자연스러운 해설을 생성합니다.
