## Mode별 Diff 섹션

`mode` 값에 따라 정확히 **하나만** 생성. 공백 섹션 금지.

### `mode: initial`
섹션 전체 생략 (제목도 출력하지 않음).

### `mode: intraday_refresh`
```markdown
## Intraday 업데이트 내역

> 직전 리포트: `{직전 파일명}` ({HH:MM} KST)

- TL;DR 변화: {직전 한줄평} → {현재 한줄평}
- 신규 캡처 티커: {Tier 2/3 신규 추가분 목록, 없으면 "없음"}
- 뉴스/breadth 업데이트: (핵심 변화만, 직전 보고 반복 금지)
```
*주의*: 지수는 동일 세션 종가라 diff 없음. "변동 없음" 공백 bullet 금지.

### `mode: next_day`
```markdown
## 전일 대비 변화

- TL;DR 변화: {전일 한줄평} → {오늘 한줄평}
- SPY: {전일 dp} → {오늘 dp}
- QQQ: {전일 dp} → {오늘 dp}
- DIA: {전일 dp} → {오늘 dp}
- IWM: {전일 dp} → {오늘 dp}
- VIX: {전일} → {오늘}
- 로테이션 라벨: {전일} → {오늘}
- 섹터 Top 3 변화: {전일 상위 3} → {오늘 상위 3}
```
